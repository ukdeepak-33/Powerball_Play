## ✨ Features

| Feature | Description |
|---|---|
| 🔥 Hot & Cold Numbers | Track which numbers are trending or overdue |
| 📊 Frequency Analysis | Full historical frequency charts per position |
| 🤖 AI Analysis | Groq-powered insights on every analysis page |
| 🗓️ Auto Scheduler | Generates picks automatically on every draw day |
| 🎯 Smart Pick Generator | AI-assisted number generation strategies |
| 🔍 Check My Numbers | Compare your picks against all historical draws |
| 📈 Pattern Analysis | Pairs, triplets, consecutive trends, odd/even splits |
| 📅 Yearly Trends | Year-by-year breakdown of white ball behaviour |
| 🧩 Custom Combinations | Build and save your own combination strategies |
| 📱 PWA Mobile App | Install as a native-feeling app on any phone |

---

## 🛠️ Tech Stack

- **Backend:** Python · Flask · APScheduler
- **Database:** Supabase (PostgreSQL)
- **Frontend:** Jinja2 · Tailwind CSS · Chart.js
- **AI:** Groq API · llama-3.1-8b-instant
- **Hosting:** Render.com
- **Mobile:** Progressive Web App (PWA)
